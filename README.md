# Optimization of Wind Turbine Performance Using Digital Twin Framework

## Overview
This project develops an AI-powered digital twin of a wind turbine
for performance simulation and optimization.

## Features
- Wind turbine performance modeling
- Machine learning-based optimization
- Streamlit-based interactive UI

## Tech Stack
- Python
- Streamlit
- NumPy, Pandas
- Machine Learning

## How to Run
```bash
pip install -r requirements.txt
streamlit run DT-STREAMLIT.py
